# 내돈백과

지원금, 월급, 세금, 노동, 가족 급여 정보를 검색하고 바로 계산까지 이어지는 생활형 검색 허브입니다.

## 현재 포함된 주요 기능

- 통합 검색과 카테고리 필터
- 검색 자동완성 형태의 추천 목록
- 빠른 프로필 입력과 저장 프로필 불러오기
- 계산기와 검색 주제 즐겨찾기
- 자격 체크 카드
- 지역별 공식 포털 묶음
- 시즌별 점검 캘린더
- 공식 기준 업데이트 카드
- 광고 위치를 고려한 레이아웃 자리

## 현재 포함된 계산기

- 월급 브리핑
- 4대보험 빠른 계산
- 최저임금 체크
- 시급 · 월급 변환
- 주휴수당
- 연장 · 야간 · 휴일수당
- 퇴직금
- 실업급여
- 육아휴직급여
- 육아기 근로시간 단축 급여
- 출산휴가급여
- 배우자 출산휴가 급여
- 고용보험 미적용자 출산급여
- 연차수당
- 휴업수당

## UX 방향

- 긴 숫자 대신 `만원` 단위와 선택 칩을 우선 사용
- 공용 프로필을 한 번 입력하면 여러 계산기에 자동 반영
- `+ / -` 버튼과 실시간 결과 갱신으로 입력 부담 최소화
- 저장 프로필과 즐겨찾기로 반복 방문 사용성 강화
- 공식 링크를 결과 카드 옆에 배치해 실제 신청 단계로 자연스럽게 연결

## 주의

- 월급 브리핑의 근로소득세는 빠른 탐색용 추정치입니다.
- 실업급여, 육아휴직급여, 출산휴가급여 등은 실제 이직 사유, 기업 규모, 신청 시기, 피보험 단위기간에 따라 결과가 달라질 수 있습니다.
- 실제 신청이나 신고 전에는 카드 안의 공식 링크에서 최신 기준을 다시 확인해야 합니다.
