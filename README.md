# 내돈백과

지원금, 세금, 월급, 노동 정보를 한 번에 찾고 바로 계산까지 이어지는 생활돈 검색 허브 MVP입니다.

## 현재 포함된 계산기

- 월급 브리핑
- 4대보험 빠른 계산
- 주휴수당
- 퇴직금
- 실업급여
- 육아휴직급여
- 출산휴가급여
- 연차수당

## UX 방향

- `만원` 단위 입력으로 긴 숫자 입력 부담 축소
- 공용 프로필을 한 번만 입력하면 여러 계산기에 자동 반영
- 칩 선택과 증감 버튼으로 모바일 입력 스트레스 완화
- 실시간 결과 갱신으로 계산 버튼 의존도 최소화

## 주의

- 월급 브리핑의 소득세 추정은 국세청 간이세액표를 완전히 재현한 값이 아니라 빠른 탐색용 추정치입니다.
- 실업급여, 육아휴직급여, 출산휴가급여, 퇴직금은 실제 자격요건과 사업장 조건에 따라 달라질 수 있습니다.
- 실제 신청과 신고 전에는 카드 내 공식 사이트 링크를 반드시 다시 확인해야 합니다.
