# 내돈백과

지원금, 세금, 월급, 노동 정보를 한 번에 찾는 생활돈 검색 허브 MVP입니다.

## 현재 구성

- `index.html`: 메인 화면
- `styles.css`: 브랜드 스타일과 반응형 레이아웃
- `script.js`: 검색, 카테고리 필터, 추천 키워드, 간편 계산기 로직

## 실행 방법

정적 사이트라서 별도 설치 없이 아래 파일을 브라우저로 열면 됩니다.
